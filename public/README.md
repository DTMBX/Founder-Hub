# Public Assets

This directory contains public-facing assets for the Founder Hub Founder Hub.

## Generate Meta Images

1. Open `/public/og-image-generator.html` in a browser
2. Download the following files:
   - `og-preview.png` (1200x630px) - for social media previews
   - `favicon-32x32.png` - browser favicon
   - `apple-touch-icon.png` (180x180px) - iOS home screen icon

3. Place all downloaded files in this `/public` directory

## Files

- **og-preview.png** - Open Graph preview image for LinkedIn/Twitter/Slack shares
- **favicon-32x32.png** - Standard favicon
- **apple-touch-icon.png** - Apple touch icon for iOS devices
- **og-image-generator.html** - Tool to generate the above images
