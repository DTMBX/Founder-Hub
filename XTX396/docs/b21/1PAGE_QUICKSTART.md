# 1-Page Quickstart — B21 Operator Mode

> Generate a professional site in under 5 minutes.

---

## Quick Steps

1. **Open Operator Mode** — Navigate to the operator dashboard.
2. **Pick a blueprint** — Select your business type (e.g., Law Firm).
3. **Fill in details** — Business name, contact info, address.
4. **Add page content** — Write or paste content for each page.
5. **Generate** — Click generate. The pipeline validates, builds, and hashes.
6. **Preview** — View the watermarked preview.
7. **Share** — Generate a secure share link (valid 24 hours).

## What You Need

| Item | Example |
|------|---------|
| Business name | "Smith & Associates" |
| Contact info | "contact@smithlaw.com" |
| Address | "123 Main St, Springfield, IL" |
| Page content | ~20+ words per page |

## What Happens Behind the Scenes

```
Your input → Validate → Scaffold → Render → Watermark → Hash → Store
```

- **Validate**: Checks all required fields.
- **Scaffold**: Builds pages from the blueprint template.
- **Render**: Generates accessible HTML.
- **Watermark**: Adds "PREVIEW" overlay.
- **Hash**: SHA-256 every file for integrity.
- **Store**: Saves the record.

## Key Rules

- All previews are watermarked. This cannot be bypassed.
- Share links expire after 24 hours by default.
- The system will not generate a site if required fields are missing.
- Every action is logged in the audit trail.
