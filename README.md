# Founder-Hub

Operations hub, portfolio, and accountability platform by Devon Tyler Barber.

## Features

- Public portfolio with pathway modes (investors, legal, about, marketplace)
- Court case tracking and transparency system
- Admin dashboard with 2FA authentication
- Client site generation and management
- Investor and offerings pages
- Contact form with honeypot spam protection

## Tech Stack

- React 19 + TypeScript
- Vite 7
- Tailwind CSS v4 + shadcn/ui
- Framer Motion
- Radix UI primitives
- Vitest + Playwright

## Quick Start

```bash
node -v          # must be 20+
npm install
npm run dev      # http://localhost:5175
```

See [docs/LOCAL-DEVELOPMENT.md](docs/LOCAL-DEVELOPMENT.md) for full setup instructions.

See [docs/FRONTEND-ARCHITECTURE.md](docs/FRONTEND-ARCHITECTURE.md) for CSS architecture.
